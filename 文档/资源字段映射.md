# 资源字段映射

这份文件用于把策划字段和资源 / 脚本字段对齐，减少后续实现时的二次解释。

## 英雄文档 -> 资源映射

- `基本信息`
  对应：`英雄定义.gd`
- `基础属性`
  对应：`英雄定义.gd` + `属性成长定义.gd`
- `专属技能 / 传承技能`
  对应：`技能定义.gd` + `技能效果定义.gd`
- `专属天赋`
  对应：`天赋定义.gd`
- `专属暗金饰品`
  对应：`装备定义.gd` + `装备词条定义.gd`

## 怪物文档 -> 资源映射

- `基本信息 / 属性 / AI 与行为`
  对应：`敌人定义.gd`
- `技能`
  对应：`技能定义.gd` + `技能效果定义.gd`
- `精英化 / 词缀规则`
  对应：`怪物词缀定义.gd` + 敌人定义中的词缀池字段
- `掉落`
  对应：后续掉落表资源

## 状态类 -> 资源映射

- `状态ID / 状态名称 / 是否负面 / 持续回合 / 最大层数`
  对应：`状态效果定义.gd`
- `每回合效果 / 属性修正列表`
  对应：`技能效果定义.gd` 和 `属性修正定义.gd`

## 特质 / 怪癖 / 疾病 -> 资源映射

- `ID / 名称 / 描述`
  对应：`特质定义.gd`、`怪癖定义.gd`、`疾病定义.gd`
- `属性修正`
  对应：`属性修正定义.gd`
- `条件修正`
  对应：`条件属性修正定义.gd`

## 随机事件 -> 资源映射

- `事件`
  对应：`随机事件定义.gd`
- `选项`
  对应：`事件选项定义.gd`
- `结果`
  当前仍按 Dictionary 列表挂在事件或选项资源中

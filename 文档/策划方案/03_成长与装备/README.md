# 成长与装备导航

这个目录负责角色长期成长、装备体系和数值参考，是 build 感和长期养成的核心。

## 这个目录回答什么

- 装备有哪些部位、品质、词条和生成规则
- 英雄升级、传承、养成如何进行
- 关键数值区间怎么设

## 文件说明

- [`01_装备与词条系统.md`](./01_装备与词条系统.md)
  装备部位、品质、词条、生成与掉落规则。
- [`02_英雄养成系统.md`](./02_英雄养成系统.md)
  升级、成长、传承、技能相关养成。
- [`03_数值平衡参考表.md`](./03_数值平衡参考表.md)
  关键数值区间和参考标尺。
- [`04_特质怪癖疾病完整资源表.md`](./04_特质怪癖疾病完整资源表.md)
  长期状态系统的完整内容表，适合直接资源化。

## 建议阅读顺序

1. [`02_英雄养成系统.md`](./02_英雄养成系统.md)
2. [`01_装备与词条系统.md`](./01_装备与词条系统.md)
3. [`03_数值平衡参考表.md`](./03_数值平衡参考表.md)

## 常见修改场景

- 改英雄成长节奏：优先改 `02`
- 改装备生成与词条库：优先改 `01`
- 改基础数值区间：优先改 `03`

## 依赖关系

- 与 [`../02_战斗系统`](../02_战斗系统/) 强相关
- 会影响英雄模板、怪物掉落和城镇强化

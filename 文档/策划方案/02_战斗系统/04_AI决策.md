5.5 AI 决策规则

攻击型：
1. 有合法 AOE 且敌方存活数 >= 3 时优先 AOE
2. 否则使用当前合法技能中预估伤害最高者
3. 目标默认选 HP 百分比最低英雄

压力型：
1. 优先带“增加压力”效果的技能
2. 若有英雄压力 >= 160，优先攻击该英雄
3. 若没有则退化为攻击型

支援型：
1. 若友方最低 HP < 35%，优先治疗
2. 若友方有负面状态，优先净化
3. 若都不满足，使用普通攻击

混乱型：
- 从全部合法技能中等权随机
- 从合法目标中等权随机

Boss AI：
- 以阶段覆盖为准
- 阶段切换后立即重算技能优先级

5.5.1 行动意图生成约束

- AI 决策服务只负责生成行动意图，不直接执行行动
- 玩家侧在 UI 中选择技能和目标后，同样生成行动意图
- 战斗控制器统一调用 `提交行动意图(意图: 行动意图)` 进行最终校验
- 若意图失效，例如目标已死、目标变尸体、站位变化，则：
  - 玩家意图：返回非法，由 UI 重新选择
  - AI 意图：自动重算一次，仍失败则改为防御性跳过

5.3 站位与位移

- 站位从前到后为 1, 2, 3, 4
- 位移是逻辑位移，不操作像素坐标
- 推拉只对存活单位生效，对尸体无效
- 若目标位被尸体占据，位移失败
- 若目标位越界，按边界截断

5.3.1 站位合法性服务

脚本：`脚本/领域/战斗/站位合法性服务.gd`

职责：
- 校验施放者是否能在当前站位施放技能
- 计算技能当前可选目标列表
- 判断位移后的最终落点是否合法
- 过滤被尸体、空位、阵营、标签限制影响的目标

接口：
- `static func 技能可施放(施放者, 技能定义, 战斗状态) -> bool`
- `static func 获取合法目标列表(施放者, 技能定义, 战斗状态) -> Array`
- `static func 计算位移结果(目标单位, 位移量, 战斗状态) -> int`
- `static func 尸体是否可选中(技能定义) -> bool`

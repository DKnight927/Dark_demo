# 战斗系统目录

本目录拆分自战斗相关章节。

文件说明：
- `01_战斗主流程`：初始化、ATB推进、回合阶段
- `02_站位与位移`：站位、目标合法性、位移校验
- `03_尸体机制`：尸体生成、阻挡、清除、边界处理
- `04_AI决策`：AI模式与行动意图约束
- `05_攻击特效_Proc`：装备特效触发与结算
- `06_伤害结算`：命中、暴击、减伤、战斗结束写回
- `07_压力与干预链`：压力来源、干预检定、崩溃后果
- `08_属性公式`：派生属性、叠加顺序、属性计算服务
- `09_Boss阶段机制`：Boss阶段切换与约束
- `10_怪物词缀系统`：敌方专属词缀、抽取规则、层级限制、效果清单

# 战斗系统导航

这个目录是战斗规则主入口。凡是回合、站位、伤害、压力、Boss、怪物词缀等问题，优先从这里找。

## 这个目录回答什么

- 一场战斗怎么开始、推进、结束
- 站位、位移、尸体如何影响行动合法性
- AI 如何生成行动意图
- 伤害、属性、压力、Boss、怪物词缀怎样结算

## 常见修改场景

- 改行动阶段：优先改 `01`
- 改站位、推拉、清尸：优先改 `02` 和 `03`
- 改伤害、暴击、减伤：优先改 `06` 和 `08`
- 改怪物精英化：优先改 `10`
- 改 Boss 机制：优先改 `09`

第一章：项目概览与核心原则

1.1 项目定位

《黑暗之源》是一款 2D 回合制地牢探索 RPG。核心体验由三部分组成：
- 局内高压探索：地牢资源有限，英雄可能死亡，站位与技能顺序决定胜负
- 局外长期经营：通过城镇设施、名册管理、资源分配和养成形成长期目标
- 情绪与代价：英雄不是纯数值单位，压力、怪癖、疾病、死亡都会留下长期后果

1.2 核心设计原则

1. 城镇域和地牢域严格分离
2. 一切规则先数据化，再写业务流程
3. 静态定义、持久状态、运行时状态严格分层
4. 服务层不持有状态，控制器协调流程，UI 只响应信号
5. 所有关键系统都必须有边界规则，禁止“运行时再想”

1.3 核心循环

每周循环：
1. 城镇管理
2. 招募 / 升阶 / 分解 / 装备 / 技能 / 天赋 / 治疗
3. 选择地牢并出征
4. 探索节点、经历战斗与事件
5. 结算战利品、经验、状态变化
6. 返回城镇并推进下一周

1.4 关键体验指标

- 战斗时长：普通战 2-4 分钟，精英战 4-6 分钟，Boss 战 6-10 分钟
- 单次远征节点数：8-12 个有效节点
- 英雄死亡率目标：前中期单次远征 5%-12%，后期 8%-18%
- 玩家主要决策频率：每 20-40 秒出现一次有代价的明确选择

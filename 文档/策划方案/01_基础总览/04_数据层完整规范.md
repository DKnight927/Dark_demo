4.2 静态定义字段

4.2.1 基础通用定义类

属性修正定义（脚本/数据/定义/属性修正定义.gd）：
- 属性: 属性类型
- 修正类型: String
- 数值: float
- 是否百分比: bool
- 叠加方式: String

约束：
- 修正类型可选：`加算`、`乘算`、`覆盖`
- 叠加方式可选：`独立叠加`、`同源取高`、`同源覆盖`
- 示例：
  - 力量 +10：属性=力量，修正类型=加算，数值=10，是否百分比=false
  - 生命值上限 +20%：属性=生命值上限，修正类型=乘算，数值=0.2，是否百分比=true

条件属性修正定义（脚本/数据/定义/条件属性修正定义.gd）：
- 条件ID: String
- 条件描述: String
- 条件类型: String
- 条件参数: Dictionary
- 属性修正列表: Array

条件类型标准值：
- `hp_below_percent`
- `stress_above_value`
- `lighting_below_value`
- `position_is`
- `target_is_elite`
- `target_hp_below_percent`

规则：
- 条件修正只在属性快照构建和实时查询时生效
- 不单独序列化到运行时状态，由属性计算服务实时评估

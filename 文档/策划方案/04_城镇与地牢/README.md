# 城镇与地牢导航

这个目录负责局外恢复和局内探索，是主循环中“出征前后”与“地牢推进”的核心部分。

## 这个目录回答什么

- 城镇设施各自负责什么
- 一次地牢探索如何推进
- 随机事件怎么配置和结算

## 文件说明

- [`01_城镇域完整设计.md`](./01_城镇域完整设计.md)
  设施职责、恢复、招募、城镇侧功能边界。
- [`02_地牢探索系统.md`](./02_地牢探索系统.md)
  探索流程、节点、遭遇和推进规则。
- [`03_随机事件系统.md`](./03_随机事件系统.md)
  事件结构、选项、结果类型和配置方式。

## 建议阅读顺序

1. [`02_地牢探索系统.md`](./02_地牢探索系统.md)
2. [`03_随机事件系统.md`](./03_随机事件系统.md)
3. [`01_城镇域完整设计.md`](./01_城镇域完整设计.md)

## 常见修改场景

- 改设施职责：优先改 `01`
- 改探索推进或节点分布：优先改 `02`
- 改事件选项和结果：优先改 `03`

## 依赖关系

- 与战役状态、地牢会话状态强相关
- 奖励与成长系统、英雄状态、战斗遭遇都会引用这里

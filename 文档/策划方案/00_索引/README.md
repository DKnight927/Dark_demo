# dark 文档拆分说明

本目录由总文档拆分而来，方便后续按单元修订。

目录约定：
- `01_基础总览`：通用原则、世界观、架构、数据层
- `02_战斗系统`：战斗相关机制拆分文件
- `03_成长与装备`：装备、养成、数值
- `04_城镇与地牢`：城镇、地牢、随机事件
- `05_工程与UI`：存档、信号、UI、路线图
- `06_英雄设定`：每个英雄一个文件
- `07_怪物设定`：每个怪物一个文件
- `08_附录参考`：附录与原稿转存

修改建议：
1. 通用规则优先改 `01_基础总览` 和 `02_战斗系统`
2. 职业专属只改 `06_英雄设定`
3. 敌人模板只改 `07_怪物设定`
4. 若正文与附录冲突，以正文规则边界为准

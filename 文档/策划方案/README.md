# 策划方案

这个目录存放《黑暗之源》的拆分式策划文档。后续如果要补系统、改数值、修英雄或怪物模板，优先在这里按类别修改，不再直接回到单一大文档里维护。

## 给人和 Agent 的阅读建议

- 想快速建立上下文：先看 [`00_索引/README.md`](./00_索引/README.md)
- 想做系统级修改：先看对应目录下的 `README`
- 想改单个职业或怪物：直接进 [`06_英雄设定`](./06_英雄设定/) 或 [`07_怪物设定`](./07_怪物设定/)
- 想找历史背景和原始补充：最后再看 [`08_附录参考`](./08_附录参考/)

## 目录结构

- [`00_索引`](./00_索引/)：入口说明、资源落地清单、命名与制作顺序
- [`01_基础总览`](./01_基础总览/)：项目概览、世界观、架构规范、数据层定义
- [`02_战斗系统`](./02_战斗系统/)：战斗流程、站位、尸体、AI、伤害、Boss、怪物词缀
- [`03_成长与装备`](./03_成长与装备/)：装备、成长、词条、数值参考
- [`04_城镇与地牢`](./04_城镇与地牢/)：城镇设施、地牢探索、随机事件
- [`05_工程与UI`](./05_工程与UI/)：存档、信号、UI 与实现路线
- [`06_英雄设定`](./06_英雄设定/)：每个英雄单独文件
- [`07_怪物设定`](./07_怪物设定/)：怪物模板与怪物词缀资源表
- [`08_附录参考`](./08_附录参考/)：原稿转存和补充参考

# 赏金猎人

## 1. 基本信息

- 职业名：赏金猎人
- 英雄ID：`shang_jin_lie_ren`
- 主属性：力量
- 定位：前中排控制收割
- 推荐站位：1, 2, 3

## 2. 基础属性

- 基础属性：
  - 血量 155
  - 魔力 90
  - 力量 22，成长 2.5
  - 敏捷 20，成长 2.0
  - 智力 12，成长 1.2
  - 攻击附加 14-18
  - 基础行动速度 5.0

## 3. 专属技能

- 技能名：`追命锁链`
- 每级效果：
  - 1级：拉目标到1号位，攻击力+20，并附加束缚2回合
  - 2级：+28，束缚3回合
  - 3级：+36，并附加暴露
  - 4级：+44，暴露更强，额外冲击伤害

## 4. 传承技能

- 技能名：`处决`
- 每级效果：
  - 1级：130%攻击力+30，若目标眩晕或束缚则×2.0，暴击+30%
  - 2级：145%+40，控制状态×2.2
  - 3级：160%+50，控制状态×2.4
  - 4级：175%+60，控制状态×2.6，击杀后自身行动进度+50

## 5. 专属天赋

- 天赋名：`猎杀本能`
- 效果：
  - 对控制状态目标伤害+25%
  - 击杀任意单位后恢复30魔力

## 6. 专属暗金饰品

- 饰品名：`赏金徽章`
- 效果：
  - 固定属性：生命+150，物攻上限+35
  - 控制增伤提升到+40%，击杀回蓝提升到50
  - 追命锁链附带赏金标记，使队友对标记目标伤害+15%

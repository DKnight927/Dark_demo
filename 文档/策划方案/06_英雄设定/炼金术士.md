# 炼金术士

## 1. 基本信息

- 职业名：炼金术士
- 英雄ID：`lian_jin_shu_shi`
- 主属性：智力
- 定位：后排 DOT 与减益专家
- 推荐站位：3, 4

## 2. 基础属性

- 基础属性：
  - 血量 110
  - 魔力 210
  - 力量 10，成长 1.0
  - 敏捷 14，成长 1.4
  - 智力 32，成长 3.0
  - 攻击附加 4-6
  - 基础行动速度 4.5
  - 初始压力抗性 5%

## 3. 专属技能

- 技能名：`剧毒瓶`
- 每级效果：
  - 1级：剧毒5回合，每回合30%魔攻真实伤害，叠3层
  - 2级：每回合40%，叠4层
  - 3级：每回合50%，并对已中毒目标附加虚弱
  - 4级：每回合60%，并附加虚弱和缓速

## 4. 传承技能

- 技能名：`燃烧弹`
- 每级效果：
  - 1级：全体60%魔攻魔法伤害 + 3回合燃烧 DOT
  - 2级：75% + 燃烧20%
  - 3级：90% + 燃烧25%
  - 4级：105% + 燃烧30%

## 5. 专属天赋

- 天赋名：`毒理大师`
- 效果：
  - 自身施加的 DOT 伤害额外+15%
  - 对剧毒目标附加魔抗-10%

## 6. 专属暗金饰品

- 饰品名：`大师的蒸馏器`
- 效果：
  - 固定属性：魔力上限+150，魔法攻击+30
  - DOT 增伤强化到+30%
  - 剧毒叠满时触发一次 200% 魔攻真实伤害的毒素爆炸

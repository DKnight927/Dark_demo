这个目录按“一个英雄一个文件”组织，适合独立迭代职业，不会互相污染。

## 修改建议

- 改某个职业，只改该职业文件
- 改所有职业共用规则，不要在这里改，回到 [`../02_战斗系统`](../02_战斗系统/) 或 [`../03_成长与装备`](../03_成长与装备/)

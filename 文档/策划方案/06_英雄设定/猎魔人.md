# 猎魔人

## 1. 基本信息

- 职业名：猎魔人
- 英雄ID：`lie_mo_ren`
- 主属性：敏捷
- 定位：中前排反法特攻
- 推荐站位：2, 3

## 2. 基础属性

- 基础属性：
  - 血量 140
  - 魔力 130
  - 力量 18，成长 1.8
  - 敏捷 29，成长 2.6
  - 智力 16，成长 1.6
  - 基础行动速度 5.5
  - 初始压力抗性 5%

## 3. 专属技能

- 技能名：`银弩穿心`
- 每级效果：
  - 1级：攻击力+20，并消耗目标50魔力，魔免目标伤害×1.5
  - 2级：+28，耗蓝75，魔免×1.6
  - 3级：+36，耗蓝100，魔免×1.7
  - 4级：+44，耗蓝125，魔免×2.0，蓝归零时额外眩晕1回合

## 4. 传承技能

- 技能名：`法力封印`
- 每级效果：
  - 1级：3回合，每回合损失30%当前魔力，无法释放需魔技能
  - 2级：35%
  - 3级：40%，持续4回合
  - 4级：50%，并使魔抗-20%

## 5. 专属天赋

- 天赋名：`魔物克星`
- 效果：
  - 对有魔力值敌人伤害+20%
  - 对魔免或高魔抗目标额外+15%

## 6. 专属暗金饰品

- 饰品名：`猎魔者之印`
- 效果：
  - 固定属性：命中+15%，物攻上限+30
  - 对有魔力值敌人的加成提升到+35%
  - 银弩穿心后追加一次短冷却追射

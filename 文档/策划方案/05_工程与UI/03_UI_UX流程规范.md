第十四章：UI/UX 流程规范

14.1 城镇界面

- 主 Hub 展示设施、资源、周数、章节
- 英雄名册显示等级、压力、怪癖、疾病、是否已服务、是否已出战
- 装备管理支持排序与筛选

14.2 战斗界面

- 左右两侧显示双方 4 槽阵型
- 尸体位用半透明倒地图像显示
- 顶部显示行动条
- 底部显示技能栏、冷却、魔力、目标提示
- 右侧显示最近 8 条战斗日志

14.3 死亡与尸体表现

- 单位死亡时立即播放死亡动画
- 随后切换为尸体图
- 尸体回合数可不直接显示数字，但鼠标悬停显示“剩余 2 / 1 回合”
- 远征结算页按死亡顺序逐一展示英雄遗言

14.4 事件界面

- 展示标题、描述、选项
- 需检定时在按钮上显示“力量 35+”之类提示
- 结果先显示文字，再结算数值变化

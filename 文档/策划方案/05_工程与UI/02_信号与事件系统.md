战斗单位构建服务：
- 负责把英雄战役状态 / 敌人定义转成战斗单位运行时
- 负责挂载天赋、装备特效、词缀、怪癖、疾病
- 负责初始化已施加状态实例列表

核心接口：
- `static func 构建英雄单位(英雄战役状态, 英雄定义, 战斗状态, 数据仓库) -> 战斗单位运行时`

- `static func 构建尸体单位(死亡单位, 战斗状态) -> 战斗单位运行时`

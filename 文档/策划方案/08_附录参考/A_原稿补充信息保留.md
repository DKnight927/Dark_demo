A.1 技术栈

- 引擎：Godot 4.6，Forward Plus 渲染
- 语言：GDScript 2.0，全中文标识符
- 渲染：Direct3D 12 为 Windows 主平台方案，Metal 为 macOS 方案
- 平台目标：Windows 主平台，Mac / Linux 次要适配

A.2 美术管线原则

- 前期允许使用 AI 占位图，只服务流程验证
- 后期正式像素资源替换时，不允许重写布局逻辑
- UI 不允许把尺寸写死到逻辑中
- Control 界面优先容器、自适应布局、相对锚点、动态缩放
- 战斗站位逻辑只认逻辑槽位，不依赖屏幕坐标

A.3 资源叙事含义

- 金币：领主家族遗留的最后财富
- 补给品：废弃仓库中搜刮出的食物和药品
- 技能书：散落于地牢中的古代知识
- 技能晶石：怪物体内提炼的魔法结晶，用于强化技能
- 天赋晶石：更稀有的异变物质，用于洗炼天赋
- 稀有材料：用于设施升级和装备强化的异变产物

A.4 服务层补充约束

- 服务脚本不继承 Node
- 方法命名采用 `动词_名词` 风格
- 除修改传入状态对象外，不直接操作外部对象

A.5 控制器层补充约束

- 控制器持有当前域状态
- 协调多个服务完成业务流程
- 状态变更后统一发信号
- 暴露给 UI 的接口需标注副作用预期

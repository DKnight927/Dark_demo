# 附录参考导航

这个目录主要用于保留历史信息、原稿转存和辅助参考，不作为第一规则来源。

## 这个目录回答什么

- 原稿里有哪些补充信息值得保留
- 命名规范和设计决策记录在哪里
- 当正文信息不够时，可以去哪里找更多背景

## 使用建议

- 正文没写清时，再回来看附录
- 如果附录与正文冲突，以正文为准
- 如果要迁移附录信息进入正文，建议在对应系统目录修改，而不是直接在附录定规则

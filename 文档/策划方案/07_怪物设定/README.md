# 怪物设定导航

这个目录按“一个怪物一个文件”组织，并单独维护怪物词缀资源表。

## 这个目录回答什么

- 每只怪的定位、技能、行为、掉落和层级
- 怪物词缀池有哪些，如何随机赋予精英怪

## 修改建议

- 改某只怪，只改该怪文件
- 新增怪物，优先按 [`00_怪物模板.md`](./00_怪物模板.md) 组织
- 改精英词缀或随机赋予规则，优先改 [`00_怪物词缀资源表.md`](./00_怪物词缀资源表.md) 和 [`../02_战斗系统/10_怪物词缀系统.md`](../02_战斗系统/10_怪物词缀系统.md)
- 改 Boss 公共规则，不要只改 Boss 文件，也要同步看 [`../02_战斗系统/09_Boss阶段机制.md`](../02_战斗系统/09_Boss阶段机制.md)

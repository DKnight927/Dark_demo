# 文档导航

这个目录放的是项目文档入口。为了后面更好找内容，可以按下面这个顺序理解：

## 常用入口

- [`策划方案`](./策划方案/README.md)
  拆分后的完整策划文档总入口，后续应优先维护这里。
- [`开发底座总览.md`](./开发底座总览.md)
  当前从策划进入正式开发前的总入口。
- [`开发阻塞清单.md`](./开发阻塞清单.md)
  当前仍会卡开发的关键缺口。
- [`开发任务总表.md`](./开发任务总表.md)
  开发顺序、依赖和验收标准。
- [`资源字段映射.md`](./资源字段映射.md)
  策划字段和资源脚本字段的对应关系。
- [`仓库地图.md`](./仓库地图.md)
  当前仓库的目录和文件总览，适合快速找文件。
- [`项目总策划案.md`](./项目总策划案.md)
  历史整稿保留版，适合全文搜索或查原始上下文。
- [`目录结构说明.md`](./目录结构说明.md)
  代码和资源目录约定说明。
- [`开工路径与首批脚本清单.md`](./开工路径与首批脚本清单.md)
  更偏“现在先做什么”的落地清单。
- [`上下文锚点.md`](./上下文锚点.md)
  适合快速回忆项目当前共识和关键术语。

## 文档怎么找

- 想看规则：进 [`策划方案`](./策划方案/README.md)
- 想快速找具体文件：看 [`仓库地图.md`](./仓库地图.md)
- 想知道现在该先补什么：看 [`开发阻塞清单.md`](./开发阻塞清单.md)
- 想按任务推进：看 [`开发任务总表.md`](./开发任务总表.md)
- 想把文档转成资源：看 [`资源字段映射.md`](./资源字段映射.md)
- 想看原始大文档：看 [`项目总策划案.md`](./项目总策划案.md)
- 想看开发先后顺序：看 [`开工路径与首批脚本清单.md`](./开工路径与首批脚本清单.md)
- 想确认项目目录规范：看 [`目录结构说明.md`](./目录结构说明.md)

## 当前约定

- 规则修订优先改拆分后的 [`策划方案`](./策划方案/README.md)
- 旧的 [`项目总策划案.md`](./项目总策划案.md) 主要用于参考，不再作为第一维护入口

# 数据导航

这个目录用于放配置和演示数据。当前已经有战斗演示相关数据目录。

## 目录地图

- [`定义`](./定义/)
  正式资源定义目录，后续 `.tres` 应优先落这里。
- [`战斗`](./战斗/)
  战斗相关配置与演示资源。

## 说明

- 如果后面开始正式建 `.tres`，建议按策划文档里的资源落地清单继续扩这个目录
- 正式资源字段和落地方向参考 [`../文档/资源字段映射.md`](../文档/资源字段映射.md)
- 配套规则入口在 [`../文档/策划方案/00_索引/资源落地清单.md`](../文档/策划方案/00_索引/资源落地清单.md)

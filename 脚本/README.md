# 脚本导航

这个目录存放 Godot 脚本。当前主要已经落地的是数据层和战斗域相关脚本。

## 目录地图

- [`应用`](./应用/)
  项目启动和全局装配。
- [`应用流程`](./应用流程/)
  主流程与战役切换逻辑。
- [`数据`](./数据/)
  `Resource` 定义、运行时状态、枚举类型、装备/技能/英雄/敌人等核心数据结构。
- [`领域/战斗`](./领域/战斗/)
  战斗控制器、结算服务、AI、站位、伤害等战斗逻辑。
- [`领域/城镇域`](./领域/城镇域/)
  城镇侧脚本骨架。
- [`领域/地牢域`](./领域/地牢域/)
  地牢探索侧脚本骨架。
- [`领域/共用`](./领域/共用/)
  存档和跨域共享逻辑。
- [`工具`](./工具/)
  后续导表、校验和批量生成脚本。

## 先看哪里

- 想看“有哪些数据类”：先看 [`数据`](./数据/)
- 想看“战斗怎么跑起来”：先看 [`领域/战斗`](./领域/战斗/)
- 想直接找某个脚本：看 [`../文档/仓库地图.md`](../文档/仓库地图.md)

1. [`数据/行动意图.gd`](./数据/行动意图.gd)
